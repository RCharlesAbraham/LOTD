1 Create Fast2SMS Account

‚ö†Ô∏è Keep your API key secure! Don't share it publicly.
5 Configure in LOTD Admin
  1. Login to LOTD Admin Panel
  2. Go to Settings page
  3. Find SMS API Configuration section
  4. Select Fast2SMS (India) from dropdown
  5. Enter your API Key
  6. Set Sender ID (6 characters, e.g., "FSTSMS")
  7. Choose Route:
    • DLT Route: For registered templates (recommended for production)
    • OTP Route: Quick OTP delivery
    • Quick Route: General purpose
  8. Click Save SMS Settings
6 Test the Integration

Use the Test SMS button in admin settings to send a test message to your phone number.

‚ùì Troubleshooting

SMS not received?
  • Check if you have sufficient balance in Fast2SMS wallet
  • Verify API key is correct
  • Ensure phone number is 10 digits (without +91)
  • Check if number is valid and active
  • Try switching from DLT to OTP route for testing
API Error?
  • Check API key has not expired
  • Verify sender ID is exactly 6 characters
  • Check Fast2SMS dashboard for error logs
  • Ensure your IP is not blocked
Check Logs

Check notification logs in admin panel: Logs ‚Üí Notification Logs

Look for error messages in the response column.
